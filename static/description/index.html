ARC POD - Print on Demand Integration

ARC POD is a comprehensive integration module for Odoo 18 Community Edition that enables seamless connectivity with leading print-on-demand services.

Supported Providers

  • Printify
  • Gelato
  • Printful

Key Features

  • Easy provider configuration
  • Secure API key management
  • Connection testing
  • Multi-provider support

Getting Started

After installation, navigate to Settings > General Settings > ARC POD to configure your preferred print-on-demand provider and enter your API credentials.
